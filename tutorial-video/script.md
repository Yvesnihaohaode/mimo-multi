# mimo2codex npm 安装教程视频脚本

总时长约 28 秒，横版 1920x1080，无旁白版本，靠大字标题、命令行和截图完成说明。

1. 0-4s：mimo2codex 是让最新版 Codex 接入小米 MiMo 的本地代理。
2. 4-12s：重点展示 `npm install -g mimo2codex`，强调 Node.js >= 18、安装后命令进入 PATH、用 `mimo2codex --version` 验证。
3. 12-18s：启动代理：设置 `MIMO_API_KEY`，运行 `mimo2codex`，看到 `listening on http://127.0.0.1:8788`。
4. 18-23s：配置 Codex：用 `mimo2codex print-config` 或 `print-cc-switch` 获取片段。
5. 23-28s：快速展示仓库文档和两个 agent 实测案例，收束为“安装、启动、切换、测试”。
